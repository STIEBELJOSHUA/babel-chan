What is Babelchan?

Babelchan is a simple text based message board based on 4chan. The main differences is that, unlike 4chan, the site has a near infinite ammount of boards. There are the main public boards listed on the homepage, and private boards that can be found by altering the url. To go to a private board, go to babelchan.org/board/boardname. Unlike reddit where subreddits have to be created by a user, all boards already exist, most of them just lack content. For more information on how to take advantage of the private board system, please read the FAQ. This website was mostly created so I could have a programming project to work on, not becuase I though the internet needed another message board. Anyway, feel free to take a look around and enjoy the site.

Please be sure to read the Rules before posting, and read the FAQ if you have any more questions about the site. Happy posting!
